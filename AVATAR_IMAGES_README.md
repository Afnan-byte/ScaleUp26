# Avatar Generator Images Needed

The Avatar Generator Modal requires the following preview images to be placed in the `/public` directory:

1. `/public/superhero.png` - Preview image for the Superhero avatar style
2. `/public/professional.png` - Preview image for the Professional avatar style  
3. `/public/medieval.png` - Preview image for the Medieval Warrior avatar style

These images should be:
- Aspect ratio: 3:4 (portrait orientation)
- Recommended size: 420px wide minimum
- Format: PNG with transparency if needed

You can replace these with actual avatar preview images or use placeholder images from a service like:
- https://placehold.co/420x560/png
- https://via.placeholder.com/420x560

Example images can be sourced from:
- AI-generated avatar samples
- Stock photo websites
- Custom designed preview mockups
